## Installation

1. Clone the repo
```bash
  git clone https://github.com/aryanGupta-09/TankStarsGame.git
```

2. Open with Eclipse, Intellij, or VSCode

3. Navigate to TankStarsGame/Final Project/desktop/src/com/mygdx/tank_stars

4. Run DesktopLauncher.java
